= Rails Templates

== INFO

So far, I've only written the "base" template, which contains:

* jquery (not the gem, just jquery)
* lesscss
* cucumber

== USAGE

Add this function, courtesy of Ryan Bates, to your bash profile.

	function railsapp {
		template=$1
		appname=$2
		shift 2
		rails $appname -m http://github.com/amiel/rails-templates/raw/master/$template.rb $@
	}
